# 面试流程

<span style="color:red;">6轮面试</span>
1. 电面 
2. 3轮技术面
3. Leader面
4. Manager面
每轮面试特点：
- 技术为主
- 编程题

面试时间：
- 3/5投简历
- 3/11微信电话面
- 3/17 3论技术和Leader
- 3/19 Manager面

- 复投面试CD
- 过面后走Offer
